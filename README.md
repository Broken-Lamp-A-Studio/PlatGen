# PlatGen

===Last informations===
 - Godot platformer game with generating terrain. Default game is done! Sometimes I will add some more blocks. Actually I'm working on Story Mode.

===Game Version=== 
 - PlatGen v.0.3

===Added elements===
 - Modes
 
===Ideas===
 - Digging.
 - Inventory.
 - Crafting.
 - Story Mode.
 
 
===Control===
 - W/S/A/D to move.
 - Control+Esc to go to main menu.
 - Space/W to jump.
